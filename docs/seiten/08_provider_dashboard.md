# Provider Dashboard (Mein Betrieb)

View-ID: `v-provider-home` · Anbieterseite

---

## Konzept

Zentrale Anbieter-Startseite: Tagesumsatz, Bestellungen, Tagesessen, Kochbuch, Wochenvorschau. Helles, aufgeräumtes Layout (kein dunkles Theme). FAB „+“ für neues Gericht (Inseratsflow).

---

## Aufbau

- **Header:** „Dein Dashboard“, Begrüßung „Hallo, [Name]“.
- **Karte:** Tagesumsatz, Bestellungen; Button „Teilen“.
- **Quick Actions:** 2 Kacheln (Tagesessen, Kochbuch).
- **Wochenvorschau:** Tag-Pills, Inhalt je Tag (Inserate planen).
- **FAB:** „+“ (unten rechts) → Inseratsflow starten.

---

## Regeln

- Layout Anbieterseite: hell (#f4f4f4), weiße Karten, dunkle Schrift.
- Kein Express-Button; keine Schiefertafeln.
- Layout getrennt von Kundenseite beibehalten.
