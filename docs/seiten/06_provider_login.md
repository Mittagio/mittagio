# Provider Login

View-ID: `v-provider-login` · Anbieterseite

---

## Konzept

Anmeldung für Anbieter: E-Mail, Passwort. Clean, helles Layout (kein dunkles Theme auf Anbieterseite).

---

## Aufbau

- **Oben:** Logo / Titel „Anbieter-Login“.
- **Mitte:** E-Mail-Feld, Passwort-Feld, Button „Anmelden“.
- **Unten:** Link „Passwort vergessen?“ (optional), Link zurück zur Kunden-App.

---

## Regeln

- Layout Anbieterseite: hell, aufgeräumt; keine Vermischung mit Kunden-UI.
- Keine Schiefertafeln.
