# Provider Profil

View-ID: `v-provider-profile` · Anbieterseite

---

## Konzept

Anbieter-Profil: Betriebsdaten, Einstellungen, Rechtliches, Logout. Sollte hell und aufgeräumt sein (konsistent mit restlicher Anbieter-UI).

---

## Aufbau

- **Header:** „Profil“ / Betriebsname, Zurück (zur Kunden-App oder Dashboard).
- **Inhalt:** Betriebsname, Adresse, Kontakt; Links zu Impressum, AGB, Datenschutz (Provider-Versionen).
- **Logout:** Button „Abmelden“.

---

## Regeln

- Layout Anbieterseite beibehalten; keine Vermischung mit Kunden-Profil.
- Bei dunklem Hintergrund (#121212) auf helles Layout umstellen (wie Dashboard/Abholungen).
