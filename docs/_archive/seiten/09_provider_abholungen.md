# Provider Abholungen

View-ID: `v-provider-pickups` · Anbieterseite

---

## Konzept

Übersicht der Abholungen (Abholnummern, Bestellungen) für den Anbieter. Helles Layout, Filter (z. B. heute/morgen), Aktionen (PDF, E-Mail).

---

## Aufbau

- **Header:** „Abholungen“, Buttons (PDF, E-Mail).
- **Subheader:** Status-Info (dynamisch).
- **Filter:** Segmented Control (z. B. Heute / Diese Woche).
- **Inhalt:** Grid/Liste der Abholungen mit Abholnummer, Gericht, ggf. Status.

---

## Regeln

- Helles Layout (#f4f4f4), weiße Karten.
- Layout Anbieterseite getrennt von Kundenseite.
